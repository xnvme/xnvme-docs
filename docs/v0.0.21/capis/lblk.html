lblk_cmd_scopy¶

int lblk_cmd_scopy(struct xnvme_dev * dev, uint32_t nsid, uint64_t sdlba, struct lblk_scopy_fmt_zero * ranges, uint8_t nr, enum lblk_scopy_fmt copy_fmt, int opts, struct xnvme_req * ret)

Submit, and optionally wait for completion of, a NVMe Copy Command.

See

xnvme_cmd_opts

Return

On success, 0 is returned. On error, -1 is returned, errno set to indicate and ret filled with lower-level status codes

Parameters

  • dev: Device handle obtained with xnvme_dev_open() / xnvme_dev_openf()

  • nsid: Namespace Identifier

  • sdlba: The Starting Destination LBA to start copying to

  • ranges: Pointer to ranges-buffer, see lblk_scopy_fmt_zero

  • nr: Number of ranges in the given ranges-buffer, zero-based value

  • opts: command options, see xnvme_cmd_opts

  • ret: Pointer to structure for NVMe completion and async. context
