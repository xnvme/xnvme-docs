Contributing#

xNVMe is hosted on GitHub where you can raise issue, start discussions, or set up a pull-request for contributing your changes.

This page contains information related to the latter. Specifically, the Contribution Process is of relevance to you and the Toolbox available for development.

In addition to GitHub then you can join Discord, there you can ping for feedback on your pull-request or ask questions.
